# Plaud Интеграция

## Пайплайн

Plaus → Zapier → n8n → Telegram + GitHub

## Как работает

1. Записываю голосовую заметку в Plaud
2. Plaud отправляет транскрипт в Zapier
3. Zapier передаёт в n8n
4. n8n сохраняет как markdown в GitHub
5. Авторазбор выделяет инсайты

## Обработка

- Учитывать text/plain (Zapier шлёт так)
- Фильтровать шумные транскрипты
- Сохранять оригинал + разбор
